Git Push

Klicke hier um mehr über Git Push zu lernen

Git Branch

Verweist auf einen bestimmten Commit kann durch „git branch“ eröffnet werden, durch „git checkout“ können diese mit dem Head „ausgewählt“ werden.

Git Clone

Der Befehl git clone dient dazu, ein vorhandenes Repository als Ziel festzulegen und einen Klon oder eine Kopie des Ziel-Repositorys zu erstellen.
